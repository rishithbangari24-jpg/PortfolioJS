

Portfolio Website
Project Overview
This is a modern, responsive portfolio website built with vanilla JavaScript, HTML5, and CSS3. The project showcases frontend development skills through an interactive single-page application featuring dark/light theme switching, smooth scrolling navigation, and a fully functional contact form with validation.

✨ Key Features:

Responsive Design: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices with custom media queries for optimal viewing experience
Dark/Light Theme Toggle: Elegant theme switching functionality with smooth transitions and persistent user preferences
Smooth Scrolling Navigation: Fixed header navigation with smooth scroll behavior and proper scroll margins to prevent content from hiding behind the header
Interactive Contact Form: Dynamically generated contact form with real-time validation, visual feedback, and success messaging
Professional Layout: Clean, modern design with card-based layouts for skills and projects sections
Form Validation: Client-side validation with visual indicators and user-friendly error handling
🛠️ Technologies Used:

HTML5: Semantic markup with proper accessibility attributes and navigation structure
CSS3: Modern styling with CSS custom properties (variables), flexbox/grid layouts, smooth animations, and comprehensive responsive design
Vanilla JavaScript: DOM manipulation, event handling, dynamic content creation, and interactive functionality without external libraries
